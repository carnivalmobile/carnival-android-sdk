getRecommendations

fun getRecommendations(sectionId: String, handler: RecommendationsHandler?): Unit

Returns the output from a Site Personalization Manager section, an array of recommendations for the given user. It is suggested you use this in conjunction with setUserEmail to identify the user to Sailthru.

Parameters

sectionId - A string representing the section ID for this recommendation

handler - an anonymous instance of RecommendationsHandler. Must implement onSuccess() and onFailure().
