handleNotification

fun handleNotification(remoteMessage: RemoteMessage): Unit

Pass received notification to Sailthru Mobile SDK. NOTE: This method should only be called if the app implements its own FirebaseMessagingService instance, in which case it must be called in order to provide FCM notifications to the Sailthru Mobile SDK for processing. Otherwise this will be handled automatically by the Sailthru Mobile SDK.

Parameters

remoteMessage - FCM notification
