fun setVibrate(pattern: LongArray!): NotificationConfig!

Set the vibration pattern to use. On some platforms, a notification that vibrates is more likely to be presented as a heads-up notification.

Parameters

pattern - LongArray!: The vibrate pattern

Return
NotificationConfig!: The same NotificationConfig.
