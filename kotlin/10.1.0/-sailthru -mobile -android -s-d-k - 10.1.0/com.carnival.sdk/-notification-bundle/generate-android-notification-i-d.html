generateAndroidNotificationID

open fun generateAndroidNotificationID(): Int

Build an id to be used to identify the notification while calling android.app.NotificationManager#notify(int, Notification). This ID is used by the Android OS to collapse notifications, if necessary.

Return
Int: hashcode of: the collapse_key when available, otherwise the Sailthru notification ID (guaranteed to be unique), and if that's not available generates a random id
