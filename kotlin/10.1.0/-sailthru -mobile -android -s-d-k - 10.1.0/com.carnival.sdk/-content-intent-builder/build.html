build

@Nullable abstract fun build(context: Context!, bundle: Bundle!): PendingIntent?

This method is called to build the intent to be executed when the notification is tapped Please set your own ContentIntentBuilder using NotificationConfig#setContentIntentBuilder(ContentIntentBuilder)

Parameters

context - Context!: The application context

bundle - Bundle!: message data as String key/value pairs.

Return
PendingIntent?: a PendingIntent to be executed or null to keep the default behaviour from Carnival
