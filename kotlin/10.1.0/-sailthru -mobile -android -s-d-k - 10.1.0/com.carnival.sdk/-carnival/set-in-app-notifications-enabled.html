setInAppNotificationsEnabled

static fun setInAppNotificationsEnabled(enabled: Boolean): Unit

Enables or disables the display of In-App Notifications. Must be called before #startEngine(Context, String). In-App Notifications are enabled by default.

Parameters

enabled - Boolean: false to disable In-App Notifications and true to re-enable.
