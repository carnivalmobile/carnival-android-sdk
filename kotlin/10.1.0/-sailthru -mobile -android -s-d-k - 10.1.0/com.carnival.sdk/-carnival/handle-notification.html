handleNotification

static fun handleNotification(remoteMessage: RemoteMessage!): Unit

Pass received notification to Carnival SDK. NOTE: This method should only be called if the app implements its own FirebaseMessagingService instance, in which case it must be called in order to provide FCM notifications to the Carnival SDK for processing. Otherwise this will be handled automatically by the Carnival SDK.

Parameters

remoteMessage - RemoteMessage!: FCM notification
