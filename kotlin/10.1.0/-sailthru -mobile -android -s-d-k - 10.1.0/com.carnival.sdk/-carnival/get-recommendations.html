getRecommendations

static fun getRecommendations(sectionId: String!, @Nullable handler: RecommendationsHandler?): Unit

Returns the output from a Site Personalization Manager section, an array of recommendations for the given user. It is suggested you use this in conjunction with #setUserEmail(String, CarnivalHandler) to identify the user to Sailthru.

Parameters

sectionId - String!: A string representing the section ID for this recommendation

handler - RecommendationsHandler?: an anonymous instance of RecommendationsHandler. Must implement onSuccess() and onFailure().
