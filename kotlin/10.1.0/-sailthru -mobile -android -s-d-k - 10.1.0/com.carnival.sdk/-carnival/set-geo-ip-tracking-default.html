setGeoIpTrackingDefault

static fun setGeoIpTrackingDefault(enabledGeoIp: Boolean): Unit

Set whether location tracking based on IP Address will be enabled or disabled by default when a device is created. This method must be called before startEngine.

Parameters

enabledGeoIp - Boolean: A boolean value indicating whether or not location based on IP Address should be enabled by default.
