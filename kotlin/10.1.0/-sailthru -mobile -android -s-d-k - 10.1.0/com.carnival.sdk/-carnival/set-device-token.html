setDeviceToken

static fun setDeviceToken(token: String!): Unit

Pass the FCM token into the Carnival SDK. NOTE: This method should only be called if the app implements its own FirebaseMessagingService instance, in which case it must be called in order to provide the FCM token to the Carnival Platform. Otherwise this will be handled automatically by the Carnival SDK.

Parameters

token - String!: FCM token provided by FirebaseMessagingService
