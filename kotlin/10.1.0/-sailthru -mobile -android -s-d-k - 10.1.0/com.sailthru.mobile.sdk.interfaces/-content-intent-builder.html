ContentIntentBuilder

interface ContentIntentBuilder : ContentIntentBuilder

Implement this interface to build a custom intent to be executed when the notification is tapped Note: Direct to URI notifications using the _u parameter will never call {@link #build(Context, Bundle)}

Example of usage directing users to SaleActivity when a notification has the custom field special_price defined.

Add your implementation using {@link NotificationConfig#setContentIntentBuilder(ContentIntentBuilder)}
