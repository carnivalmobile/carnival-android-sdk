trackClick

fun trackClick(sectionId: String, url: URI, handler: TrackHandler?): Unit

Asynchronously registers with Sailthru SPM that a section has been clicked/tapped on, transitioning the user to a detail view

Parameters

sectionId - the ID of the section to track a click for. Must not be null.

url - the URL of the detail being transitioned to. Must not be null.

handler - an instance of TrackHandler. May be null, in which case onSuccess and onFailure callbacks won't be fired.
